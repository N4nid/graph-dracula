Anchor

Variables

  • baseObject

      • The javafx object, which is dynamically resized.

  • relateToObject

      • The javafx object, which the Anchor relates to.

  • offsetVec

      • This defines the difference between the baseObject and the relateToObject.

  • keepX/Y

      • Stores, if the x or y value of the baseObject should be kept.

  • Type

      • Stores the type of Anchor. It can be "scale": relats the scale of both objects together, "pos": releates the position of them, "scale->pos": relates the scale of the releateToObject to the position of the baseObject or "pos->scale": relates the position of the releateToObject to the scale of the baseObject

Methods

  • applyAnchor()

      • Apllies the Anchor based on the defined variables.

  • applyAnchors()

      • Calls applyAnchor() function for all Anchors in a given Anchor list.
