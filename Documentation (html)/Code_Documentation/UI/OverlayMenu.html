OverlayMenu

This class represents a sub-menu with different options. It hides, when you click somewhere else. It executes the selected option via the menuHaver interface.

Relevant Variables

  • options

      • Stores available MenuOptions of the OverlayMenu.

  • menuHaver

      • This stores a refrence to any object, that can execute options from this menu.

  • offsetVec

      • This defines the difference between the baseObject and the relateToObject.

  • optionDimentions

      • This variable stores the length and height, one option in the menu should have.

Methods

  • hide()

      • This hides this menu, if you click somewhere else.
      This function is part of the Hideble interface.

  • setMenuPreset()

      • This initializes the MenuOptions for different default menus.
      Available Presets:
      euqationElement: This is the default OverlayMenu for EquationVisElements.
      It has the options edit or delete, which are executed from the ApplicationController class.
