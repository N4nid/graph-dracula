Hideble

This interface is intended to provide a standard for UI-elements, who need to disappear, when you click somewhere else.
This makes it possible to group such classes in Arrays and bulk hide them. In this case, this happens within the ApplicationController class.
This functionality is required, because users exepct a right click window for example, to go away after they interact with something else.

Methods

  • hide()

      • This method is intended to be used to hide() Hideble UI windows.
      This would be something like a right click menu or the ColorPickerWindow from the RoundColorPicker class.
