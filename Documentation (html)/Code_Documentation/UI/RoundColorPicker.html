RoundColorPicker

This class is instantiated either as a part of an EquationVisElement or as the mainColorPicker of the ApplicationController class.
It contains a the clickable button to choose the color and the context menu, to choose a color.

Relevant Variables

  • isUp

      • This determines, if the RoundColorPicker opens it's UI window over the displayButton.
      If it opens the window over the button it's true (that would mean it's the MainColorPicker of the ApplicationController class.

  • selectableColors

      • This Color array stores all possible Colors to select in the colorPickerWindow.
      If you choose a color, an index from this array is passed on.

  • displayButton

      • This is the round button, which displays the selected color and you can click to open the ColorPickerWindow.

  • colorPickerWindow

      • This is the window, that pops up, when you click the displayButton.
      It displays the selectable ColorPickButtons and has an attached Label and a closeButton.

Methods

  • Constructor

      • Set's uo the UI for the button and the window to choose colors. The window is hidden by default.

  • recalcExtraPostions()

      • This method is called, when you scroll or resize the window.
      It recalculates the positon of the colorPickerWindow if it is currently visible.

  • hide()

      • This method is called, when you click somewhere on the scene.
      It hides the colorPickerWindow, if you din't click on the buttun, or on the pane,
      because then you interacted with the UI and didn't want the colorPickerWindow to go away.
      This is part of the Hideble interface.

  • limitPos()

      • This method makes shure, the colorPickerWindow doesn't go over the window border.

  • moveTo()

      • This method moves a Node to a given postion.

  • onClick()

      • This method is called, when you click the displayButton.
      It shwos or closes the colorPickerWindow and makes shure everything is a the right place.

  • pickColor()

      • This method is called, when you click a ColorPickButton .
      It changes the associated values from this class and the color of the displayButton.

  • toRGBCode()

      • Converts a given javafx Color to a String of it's hexRGB code.

  • setupColors()

      • This method add all Colors, which i typed in manually to the selectableColors array.
