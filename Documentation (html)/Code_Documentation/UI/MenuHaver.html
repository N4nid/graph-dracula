MenuHaver

This interface is intended to provide a standard for elements, that need to be able to execute functionility from their sub menu.
This makes it possible to have a standard menu class (in this case, this would be OverlayMenu),
which can execute the selected option in any class, they are attached to.

Methods

  • executeMenuOption()

      • This method gets a String as an input instruction and is supposed to execute this input.
