Main

Variables

  • scene

      • Is the base application scene. The base fxml-file, which contains the base UI, and the main ccs file for styling is loaded into it at the start of the application.

Methods

  • start()

      • Initializes the application. It loads the fxml-file and the main css file for styling. It also initilizes a MouseEvent, that fires every time, the mouse is clicked on the main window.

  • MOUSE_PRESSED event

      • This initilizes the hideRedundantElements() method of the ApplicationController, which hides any sub menus currently opened

  • testEquationFunctions

      • Theese are some different test EquationTrees, which can be used to test for example a renderer.
