EquationVisElement

This class is instantiated inside the main ScrollPane, every time you add an eqution. It displays the equation, has a RoundColorPicker to change the color and you can right click it to delete or edit it.

Variables

  • main UI elemts

      • Stores the required UI Elements

  • equation

      • Stores the EquationTree, which represents the equation from the equationText.

  • equationText

      • Stores the text of the equation, which is displayed visually.

  • controller

      • Stores a refrence to the ApplicationController .

Methods

  • Constructor

      • Sets up the main UI and assigns some inputed variables.

  • updateTransform()

      • Calls recalcExtraPostions() of RoundColorPicker . This get's called, when UI is rescaled or you scroll on the ScrollPane and enshures, the colorPickerWindows of the associated Color pickers are always at the right positon.

  • executeMenuOption()

      • Gets called, when you choose something in the right click menu of the EquationVisElement . This function is part of the MenuHaver interface.

  • setEquationText()

      • Sets EquationText String value and updates the Label. This is called, when you apply an edit to a function.
