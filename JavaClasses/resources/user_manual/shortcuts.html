Shortcuts
//shortcut-list

General

- CTRL + W: Close Application

Text Input

- CTRL + F: Focus input field and add a new function (f(x)=)
- CTRL + L: Go to the end of the inputted text
- CTRL + E: Open/Close the expand menu
- ENTER: Add input as an equation or confirm an equation edit

Editing functions

- Double click on a visual equation to edit the clicked equation
