1. What is graph_dracula?

Graph_dracula is a simple and user-friendly application, that can visualize graphs for functions, equations and parametric functions. It was written exclusively in Java, with the help of the JavaFX libary for the user interface.

2. Who made graph_dracula?

This project was created by 3 high school students in Germany for a 3-month school project.
Here are the links to the creators personal GitHub pages:
- N4nid (responsible for the stringParser): github.com/N4nid
- ARacer06 (responsible for the equationRender, parametricRenderer and some shortcuts):
  github.com/ARacer06
- MortySmith (responisble for the UI,equationBackend,functionRenderer and the conditions):
  github.com/Morty0Smith

Every line of code and every graphic used for this project is exclusively our own work, with a few exceptions:
