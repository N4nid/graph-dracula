Shortcuts

//shortcut-list

1. General

- CTRL + SHIFT + R: Enable or disable dynamic high quality rendering.
  This setting makes the render automatically increase the detail, when you zoom out,
  which helps with functions, like tan(x) or sin(x)^-1 to be displayed more accurately.
- CTRL + W: Close Application.

2. Text Input

- CTRL + F: Focus input field and add a new function (f(x)=).
- CTRL + L: Go to the end of the inputted text.
- CTRL + E: Open/Close the expand menu.
- ENTER: Add input as an equation or confirm an equation edit.

3. Editing functions

- Double-click on a visual equation to edit the clicked equation.
- ESCAPE while editing an equation: Undo current edit and leave edit mode.
