Example Equations

This list provides some interesting examples for equations, which you can visualize with
graph_dracula to give you a better idea on first: How inputs have to be formatted
and about what is possible with our application.

//example-equation-list

1. Functions

- sin(x);if(sin(15x)<0.5) //dashed sine-wave

2. Equations

- x^2+y^2=9 //circle
- y^2=x^3-3x+1 //elyptic-curve

3. Parametric Functions

- f(t->xy):x=(sin(t)*(e^cos(t)-2cos(4t)-sin(t/12)^5));y=(cos(t)*(e^cos(t)-2cos(4t)-
  sin(t/12)^5);for(0<t<12*pi) //butterfly-curve
- g(t->xy):x=(cos(t));y=(sin(2t)/2);for(-5<t<5) //infinity-sign
- h(t->xy):x=(4cos(t)*sin(at));y=(4sin(t)*sin(at));for(0<t<2*pi) //flower
